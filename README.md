# Portal RT PWA

Struktur proyek:
- `backend/` API Node.js (Tahap 0 aktif)
- `frontend-vite/` frontend ringan React + Vite (Tahap 1)
- `frontend/` template Next.js (disimpan sebagai referensi)

Cara menjalankan backend:

```bash
cd backend
npm install
npm run start
```

Cara menjalankan frontend:

```bash
cd frontend-vite
npm install
npm run dev
```

Dokumen rancangan utama: `RANCANGAN_PORTAL_RT.md`.
