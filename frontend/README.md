# Frontend Portal RT (Tahap 1)

Frontend ini menggunakan template Modernize (Next.js + MUI) dan disesuaikan untuk Tahap 0-1.

## Cara menjalankan

```bash
npm install
npm run dev
```

Buka `http://localhost:3000`.

## Konfigurasi API

Default API: `http://localhost:3000`.
Ubah lewat env:

```bash
set NEXT_PUBLIC_API_BASE_URL=http://localhost:3000
```

## Ruang lingkup Tahap 1

- Halaman login
- Halaman registrasi warga
- Dashboard admin RT
- Daftar pendaftaran warga + detail
- Profil RT + kode undangan
